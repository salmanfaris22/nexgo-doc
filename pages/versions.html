v1.2.0 Latest
April 2026

SEO, Streaming SSR, ISR, Worker Pools & Advanced Features

Production-ready features: SEO meta tags, sitemaps, streaming SSR, incremental static regeneration, worker pools, response caching, parallel data loading, and template caching.

New Features

New
SEO Package

Meta tags, OpenGraph, Twitter Cards, sitemap.xml, robots.txt, and JSON-LD structured data generation.

New
Streaming SSR

Send HTML progressively as it's ready — show the shell immediately, stream content later. Includes SSE support.

New
Incremental Static Regeneration (ISR)

Cache pages and rebuild them in the background after a configurable interval — like Next.js ISR.

New
Worker Pools

Run tasks concurrently with controlled parallelism. Map, ForEach, and global pool support.

New
Response Caching

Cache GET responses in memory with TTL. Automatic X-Cache: HIT/MISS headers.

New
Parallel Data Loading

Load multiple data sources concurrently and merge results — like Promise.all() in JS.

New
Template Caching

Templates compiled once and cached. Parallel template loading with worker pool.

v1.1.0
April 2026

HTMX Support, State Management & API Helpers

Built-in helpers for writing shorter, cleaner API handlers with HTMX support and easy state management.

Features

New
HTMX Helpers

IsHTMX, HTMXHTML, HTMXTrigger, HTMXHeader — send HTML fragments and fire client events.

New
State Management

Thread-safe global state with SetState, GetState, DeleteState, and NewState for isolated sessions.

New
HTML Helpers

Escape() for XSS-safe output and HTML() for raw HTML responses.

New
Todo App Example

Complete HTMX-powered todo app in the docs with CRUD operations.

v1.0.5
December 2024

Initial Public Release

The Go-powered web framework inspired by Next.js. File-based routing, SSR, and hot reload.

Features

Orig
File-Based Routing

Automatic route discovery from pages/ directory with dynamic segments.

Orig
Server-Side Rendering

SSR, SSG, and SPA modes with data loaders like getServerSideProps.

Orig
Hot Reload (HMR)

SSE-based file watching with instant browser refresh.

Orig
API Routes

REST endpoints with method routing, JSON helpers, and pagination.

Orig
DevTools Panel

Built-in debug panel with route inspector, request logs, and performance metrics.

Orig
Middleware System

Recover, Logger, SecurityHeaders, Gzip, CORS, and Cache middleware.
