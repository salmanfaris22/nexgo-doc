v2.0.2 Latest
April 2026

Auth, Database, WebSockets, Cluster Mode & Full-Stack DX

Major release: JWT auth, sessions, CSRF protection, database with ORM, WebSockets, cluster mode, plugin system, server actions, i18n, image optimization, deployment adapters for 8 platforms, and much more.

Security

New
JWT Auth

Built-in JWT authentication with sign, verify, and refresh token support.

New
Sessions

Cookie-based and store-backed session management with configurable expiry.

New
CSRF Protection

Automatic CSRF token generation and validation middleware.

New
Rate Limiting

Per-route and global rate limiting with sliding window algorithm.

Data

New
Database (JSON + SQL)

Unified database layer supporting both JSON file storage and SQL databases.

New
ORM

Lightweight ORM with model definitions, relations, queries, and migrations.

New
Migrations

Schema migrations with up/down support and automatic version tracking.

New
Redis Cache

Redis-backed caching with TTL, invalidation, and pub/sub support.

Infrastructure

New
WebSockets

Native WebSocket support with rooms, broadcasting, and auto-reconnect.

New
Cluster Mode

Multi-process clustering with shared state and automatic load balancing.

New
Deployment Adapters (8 platforms)

One-command deploy to Vercel, Fly.io, Railway, Render, AWS, GCP, Azure, and Docker.

New
Health Checks

Built-in /healthz and /readyz endpoints with custom check registration.

DX

New
Plugin System

Extensible plugin architecture with lifecycle hooks and dependency injection.

New
Server Actions

Define server-side mutations directly from templates with automatic form handling.

New
Testing Utilities

Test helpers for routes, middleware, and handlers with request mocking.

New
Structured Logging

JSON-structured logging with levels, context fields, and log rotation.

New
Metrics

Prometheus-compatible metrics endpoint with request duration, error rates, and custom counters.

Frontend

New
i18n/RTL

Internationalization with locale detection, translation files, plurals, and RTL layout support.

New
Image Optimization

Automatic image resizing, format conversion (WebP/AVIF), and lazy loading.

New
Asset Pipeline

CSS/JS bundling, minification, fingerprinting, and cache-busting.

New
Error Pages/Boundaries

Custom error pages (404, 500) and error boundaries that catch panics gracefully.

New
Per-Route Middleware

Attach middleware to individual routes or route groups with chaining support.

New
CSP

Content Security Policy header generation with nonce support for inline scripts.

New
Env Variables

Typed environment variable loading from .env files with validation and defaults.

v1.2.0
April 2026

SEO, Streaming SSR, ISR, Worker Pools & Advanced Features

Production-ready features: SEO meta tags, sitemaps, streaming SSR, incremental static regeneration, worker pools, response caching, parallel data loading, and template caching.

New Features

New
SEO Package

Meta tags, OpenGraph, Twitter Cards, sitemap.xml, robots.txt, and JSON-LD structured data generation.

New
Streaming SSR

Send HTML progressively as it's ready — show the shell immediately, stream content later. Includes SSE support.

New
Incremental Static Regeneration (ISR)

Cache pages and rebuild them in the background after a configurable interval — like Next.js ISR.

New
Worker Pools

Run tasks concurrently with controlled parallelism. Map, ForEach, and global pool support.

New
Response Caching

Cache GET responses in memory with TTL. Automatic X-Cache: HIT/MISS headers.

New
Parallel Data Loading

Load multiple data sources concurrently and merge results — like Promise.all() in JS.

New
Template Caching

Templates compiled once and cached. Parallel template loading with worker pool.

v1.1.0
April 2026

HTMX Support, State Management & API Helpers

Built-in helpers for writing shorter, cleaner API handlers with HTMX support and easy state management.

Features

New
HTMX Helpers

IsHTMX, HTMXHTML, HTMXTrigger, HTMXHeader — send HTML fragments and fire client events.

New
State Management

Thread-safe global state with SetState, GetState, DeleteState, and NewState for isolated sessions.

New
HTML Helpers

Escape() for XSS-safe output and HTML() for raw HTML responses.

New
Todo App Example

Complete HTMX-powered todo app in the docs with CRUD operations.

Initial Public Release

The Go-powered web framework inspired by Next.js. File-based routing, SSR, and hot reload.

Features

Orig
File-Based Routing

Automatic route discovery from pages/ directory with dynamic segments.

Orig
Server-Side Rendering

SSR, SSG, and SPA modes with data loaders like getServerSideProps.

Orig
Hot Reload (HMR)

SSE-based file watching with instant browser refresh.

Orig
API Routes

REST endpoints with method routing, JSON helpers, and pagination.

Orig
DevTools Panel

Built-in debug panel with route inspector, request logs, and performance metrics.

Orig
Middleware System

Recover, Logger, SecurityHeaders, Gzip, CORS, and Cache middleware.
